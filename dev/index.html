individual.jl

A Julia package for specifying and simulating individual based models (IBMs), which relies on Catlab.jl, especially attributed C-Sets to create schemas which can represent a broad class of IBMs useful for epidemiology, ecology, and the computational social sciences. It is inspired by the R software individual.
